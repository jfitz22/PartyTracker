Core product requirements:
This app is for a tabletop RPG party.

Players need to be able to:
- open the app on phone or laptop
- find their character
- view and update inventory
- move items between equipped, bag, and storage
- edit item descriptions

- create and edit journal entries
- view a quest log

The DM needs to be able to:
- view all characters
- open and edit any character
- add, edit, move, or remove items
- adjust charges
- manage quests
